Aqui contém vários exemplos que são utilizados em aulas: Sistemas Operacionais I, Linguagem de Marcação, Banco de Dados, Linguagem de Programação e Front-End.

No caso de Sistemas Operacionais I, tem alguns simuladores:
  - Alocação de Recursos (HTML)
  - Gerenciador de Arquivos (HTML)
  - Tabelas de Alocações (em C, o segundo é colorido)
  - Sistema Gerenciador de Arquivos - Permissões - (em C)
  - Sistema Gerenciador de Arquivos (HTML)
  - Sistema Gerenciador de Dispositivos (HTML)

Estes simuladores foram desenvolvidos em conjunto com a IA (ChatGPT, Gemini e Claude), é um trabalho de desenvolvimento coletivo com o objetivo de deixar visível operações que, até então, são transparentes para o usuário. 

No caso das demais disciplinas, são exemplos utilizados em sala de aula para demonstrações e atividades de funcionalidades junto aos alunos.
